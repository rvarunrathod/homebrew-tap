# Rvarunrathod Tap

## How do I install these formulae?

`brew install rvarunrathod/tap/<formula>`

Or `brew tap rvarunrathod/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rvarunrathod/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
